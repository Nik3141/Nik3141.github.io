How I Created The Site


Content

I used Let's Encrypt to get an SSL Certificate for the website, which although unnecessary (because there is no potentially sensitive information being sent between the client and the server) seemed like a good idea so browsers wouldn't throw up warnings every time anyone tried to visit. I primarily used Notepad++ to write the site, testing in Chrome (which is why the scrollbar might be messed up in alternative browsers), and using w3schools for reference. I made the custom scrollbar using information from this site.
Tip: To see the source code of the website, hit Ctrl + U.
